Nacos操作指南

Posted on 2020-09-04

Nacos Server 监控

Nacos本身是一个Spring Boot 应用，

Actuator -> /nacos/actuator/prometheus

主流是采用Prometheus进行数据采集

数据持久化

面板

Grafana

可视化面板

datasource->监控数据的来源

es/ prometheus/influxdb/zipkin…

接下来采用Nacos + Prometheus + Grafana实现应用监控

  1. 安装nacos server

  2. 进入到nacos根目录下,修改./conf/application.properties配置文件

  3. 重启nacos，浏览器打开http(s)://{ip:port}/nacos/actuator/prometheus,如果能正常看到

说明配置生效

  1. 安装Prometheus

  2. 进入到Prometheus目录下，修改./prometheus.yml

  3. 启动Prometheus应用

    访问9090端口如果出现Prometheus界面则启动成功

    到这里为止其实数据监控已经完成，Prometheus可以直接展示监控到的应用数据。但是为了更强大的展示效果。所以接下来启动Grafana

  4. 安装Grafana

  5. 启动Grafana

    docker 启动

    原生启动

    访问3000端口登录，默认账号，密码：admin,admin

    进入配置 Configuration/Datasource，即刚刚配置的Prometheus

可以在官网下载相应的Dashboard配置json导入，也可以自己手动配置

done！
